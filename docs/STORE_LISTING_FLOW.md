# Store Listing DM Flow

なごとしゃの初回無料掲載DMに店舗から返信が来たときの対応フローです。無料掲載でも、画像許可・事実確認・掲載前確認を必ず取り、読者にも店舗にも誠実な記事にします。

## 1. 最初の返答テンプレ

```text
ご返信ありがとうございます。
名古屋の店舗・おでかけ情報を紹介している「なごとしゃ編集部」です。

現在、掲載実績づくりのため、初回無料で店舗紹介記事を作成しています。
掲載にあたり、店舗情報と写真素材をいくつかご共有いただけますでしょうか。

記事公開前には、店舗名・住所・営業時間・写真出典などの事実確認をお願いできます。
掲載費用は初回無料ですので、まずはお気軽にご相談ください。
```

## 2. 店舗からもらう情報チェックリスト

- 店舗名
- 住所
- 営業時間
- 定休日
- オープン日
- おすすめ商品 1〜3点
- 価格
- 公式Instagram
- 公式サイト
- GoogleマップURL
- 写真 3〜5枚
- 画像出典として表記する名称

## 3. 画像許可テンプレ

```text
ご提供いただいた写真は、なごとしゃの記事内および記事サムネイルとして使用してもよろしいでしょうか？
画像出典は「画像提供: 店舗名」の形で明記します。

使用してよい写真、使用を避けたい写真、トリミング不可などの条件があれば、あわせて教えてください。
```

## 4. 掲載前確認フロー

1. 店舗から情報を受領する
2. 写真の使用許可と出典表記を確認する
3. 記事下書きを作成する
4. 店舗名・住所・営業時間・定休日・価格などの事実確認を行う
5. 画像出典の表記を確認する
6. 掲載前に店舗へ確認を依頼する
7. WordPressで公開する
8. `app.nagotosha.com` で表示確認する
9. 掲載URLを店舗へ送付する

## 5. 掲載URL送付テンプレ

```text
掲載確認ありがとうございました。
なごとしゃでの記事公開が完了しました。

記事URL:
https://app.nagotosha.com/article/xxxx

内容に修正が必要な箇所があれば、お気軽にご連絡ください。
Instagram等でご紹介いただけると、とても嬉しいです。

このたびはご協力ありがとうございました。
```

## 6. 注意事項

- Instagram投稿画像を勝手に転載しない
- 店舗提供画像、または明確に利用許可を取った画像だけを使う
- 出典表記を必ず入れる
- 営業時間・定休日・価格・メニュー内容は変更される可能性があるため、記事内に最新情報確認の案内を入れる
- 無料掲載でも、PR・提供関係がある場合は読者に分かりやすくする
- 店舗への連絡が無理な有料営業に見えないようにする
- 画像の人物写り込み、ロゴ、キャラクター、店内掲示物などの権利リスクに注意する

## 7. 掲載対象外にする場合

次のいずれかに該当する場合は、無理に記事化せず保留または掲載対象外にします。

- 画像許可が取れない
- 公式情報が不明
- 営業状況が不明
- なごとしゃの読者と合わない
- 掲載内容の確認が取れない
- 写真の出典表記や利用範囲が曖昧
- 読者が誤解しそうな表現を修正できない

## 8. 運用メモ

- DMでは「無料掲載協力のお願い」として伝え、売り込み感を出しすぎない
- 最初から有料メニューを強く出さない
- 写真提供が難しい場合は、無理に記事化せず候補リストへ戻す
- 記事公開後は、修正連絡を受けられる状態にしておく
- 掲載実績が増えたら、/partner の掲載実例にも反映を検討する
